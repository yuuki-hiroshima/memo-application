# memo-application

# 📝 Flask メモアプリ

Flask を使用して作成した「シンプルで使いやすいメモ管理アプリ」です。  
カテゴリ管理、プライベートメモ機能、パスワード保護など、  
実用レベルの機能を備えた Web アプリケーションとなっています。

学習成果の集大成として **フルスクラッチで構築** しており、  
バックエンド処理・データ管理・UI/UX 改善まで一通り経験しています。

---

## 🚀 主な機能

### ■ メモ管理
- メモの新規作成・編集・削除
- メモは JSON 形式で保存（簡易データベース）
- 本文プレビューを一覧に表示
- 作成日・更新日をフォーマットして表示  
  （例：`2025年 11月17日 14時53分`）

### ■ カテゴリ機能
- カテゴリ別表示
- カテゴリ作成・名前変更・削除
- 「未分類」は固定カテゴリとして常に一番上に表示
- メモのカテゴリ一括変更（バルク操作）
- メモに存在するカテゴリも自動検出して反映

### ■ プライベートメモ機能（パスワード保護）
- 任意のメモをプライベート扱いできる
- プライベート画面はパスワード入力で開閉
- パスワードはユーザー設定可能
- セッションで状態管理し、一定条件でロック

### ■ UI/UX（こだわりポイント）
- タイトル行の右側に操作ボタンを配置したヘッダー
- 一覧表示は最大15件までのスクロール領域
- 画面幅が狭い場合は操作ボタンが縦並びになるレスポンシブ設計
- 入力欄・ボタン・テーブルの見た目を統一して使いやすさを改善

## 🎨 UI の特徴

- 落ち着いたライトテーマ
- 余白と行間を広く取り、読みやすさを重視
- 操作ボタンの色は用途別に分類  
  - Primary（作成など）  
  - Danger（削除）  
  - Secondary（プライベート関連）  
- テーブル行は hover でハイライト  
- 入力フォームは統一されたデザインで使いやすい

## 🧪 テスト済みシナリオ

- メモの作成 → 一覧反映 → 編集 → 削除
